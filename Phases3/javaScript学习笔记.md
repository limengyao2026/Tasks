# JavaScript 学习笔记
## 一、JS 基础概念
JavaScript 网页交互脚本语言，实现动态效果、事件处理、数据存储。

## 二、基础语法
- let / const 变量声明
- 字符串、数字、数组、对象基础数据类型
- console.log 控制台调试

## 三、DOM操作核心
- getElementById 根据ID获取元素
- querySelector / querySelectorAll 选择器获取
- 修改 innerText、value 变更页面内容

## 四、页面事件
- click 点击事件
- input 输入监听
- scroll 页面滚动监听
- 事件委托：批量处理同类子元素事件

## 五、定时器
- setInterval 循环重复执行（时钟/轮播）
- setTimeout 延迟单次执行

## 六、数据持久化
- localStorage 本地存储，刷新数据不丢失

## 七、数组遍历
- 数组 + forEach 遍历渲染列表结构
